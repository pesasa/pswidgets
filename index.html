Pswidget examples

Here are couple of examples, how to use jquery.pswidget.js.

Existing radio buttons

Applying psradio to existing radio buttons with labels

<form> <div id="radio1"> <input type="radio" name="psradio1" value="red" id="psradio1red" /> <label for="psradio1red">red</label><br /> <input type="radio" name="psradio1" value="orange" id="psradio1orange" /> <label for="psradio1orange">orange</label><br /> <input type="radio" name="psradio1" value="yellow" id="psradio1yellow" /> <label for="psradio1yellow">yellow</label><br /> <input type="radio" name="psradio1" value="green" id="psradio1green" checked="checked" /> <label for="psradio1green">green</label><br /> <input type="radio" name="psradio1" value="blue" id="psradio1blue" /> <label for="psradio1blue">blue</label><br /> </div> </form> <script type="text/javascript"> jQuery('#radio1').psradio({colors: [ 'red', 'orange', 'yellow', 'green', 'blue' ]}); </script>





Empty div-element

Applying psradio to an empty div-element creates needed radio buttons and labels first.

<form> <div id="radio2"> </div> </form> <script type="text/javascript"> jQuery('#radio2').psradio({ values: [ 'red', 'green', 'blue' ], labels: [ 'Punainen', 'Vihreä', 'Sininen' ], colors: [ 'red', 'green', 'blue' ] }); </script>

Without label-list in options, values are used for labels.

<form> <div id="radio3"> </div> </form> <script type="text/javascript"> jQuery('#radio3').psradio({ values: [ 'red', 'green', 'blue' ], colors: [ 'red', 'green', 'blue' ] }); </script>
